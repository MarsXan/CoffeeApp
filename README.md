# About
CoffeeApp is a mobile application that allows users to order coffee from a coffee shop. It is built with SwiftUI and Firebase. The application is still in development.


# Technologies and Tools
- SwiftUI
- Firebase
- Xcode 14.3
- Swift 6.4
